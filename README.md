# LMB Toolkit for MECC 2024

This repo contains MATLAB code to accompany the publication entitled 
"Estimating the Values of the PDE Model Parameters of Rechargeable 
Lithium-Metal Battery Cells using Linear EIS" at the Modeling, Estimation 
and Control Conference 2024.

- `DEMO/demoSimFOM.m`: Simulates the full-order Newman model in COMSOL (**requires COMSOL with MATLAB LiveLink**).
- `DEMO/demoTF.m`: Demonstrates the usage of the toolkit for computing frequency-response of electrochemical variables for LMB cells.
- `MSMR_REGRESSION/demoFitMSMR_NMC622.m`: Verifies the MSMR model regression using synthetic data for an NMC-622 electrode.
- `TF_REGRESSION/`: Verified the TF model regression using synthetic data.
